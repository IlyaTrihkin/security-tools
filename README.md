# security-tools

Сборник скриптов для автоматизации задач информационной безопасности.

## Содержание

- **`check_os_updates.py`** — проверяет и устанавливает доступные обновления пакетов в системах на базе apt (Debian/Ubuntu/Astra) или yum (CentOS/RedOS). Умеет выводить результат в текстовом или JSON-формате. Поддерживает автоматическую установку с повторной проверкой до полного обновления системы.

## Установка и использование

### Требования
- Python 3.6+
- Менеджер пакетов apt или yum
- Права sudo (для установки обновлений)

### Запуск

Склонируйте репозиторий или скачайте файл.

```bash
python3 check_os_updates.py [--format text|json] [--update] [--yes]

Параметры:

--format, -f — формат вывода: text (по умолчанию) или json.

--update, -u — установить все доступные обновления (циклически, пока есть).

--yes, -y — автоматически соглашаться на установку (без запроса подтверждения).

Примеры:

# Проверить обновления (текстовый вывод)
python3 check_os_updates.py

# Проверить обновления (JSON)
python3 check_os_updates.py --format json

# Установить все обновления с подтверждением
python3 check_os_updates.py --update

# Установить все обновления автоматически (без вопросов)
python3 check_os_updates.py --update --yes


Пример вывода (JSON):

[
  "openssl",
  "bash",
  "systemd"
]


Пример установки обновлений:

$ python3 check_os_updates.py --update --yes
Найдено 72 пакетов для обновления.
Устанавливаю 72 обновлений через apt...
